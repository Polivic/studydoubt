# 📚 StudyDoubt

Aplicação para organizar e acompanhar dúvidas de estudo de forma simples e eficiente.

---

## 🚀 Funcionalidades

- Criar dúvidas
- Listar dúvidas
- Marcar como resolvida ou pendente
- Excluir dúvidas
- Filtrar por status (todas, pendentes, resolvidas)
- Visualizar resumo (total, pendentes, resolvidas)

---

## 🛠 Tecnologias utilizadas

- React
- TypeScript
- Vite
- JSON Server (API fake)
